Программа генерации тестов для синтаксического анализатора

src/table-builder - директория программы стадии анализа входной грамматики и построения для неё таблицы разбора.

src/test-builder - директория программы построителя графов и генератора тестов.

Тестовые грамматики, подаваемые на вход, находятся в папке src/test-builder/grammars.

Запуск генератора тестов производится из папки src/test-builder через консоль:
node generateTests [путь до файла грамматики] [флаг режима отладки]

Пример:
node generateTests grammars/simple.txt true # выполнится генерация тестов с отображением всей технической информации при построении и обходе графа.

В результате работы генератора для заданной грамматики в папке src/test-builder/tests будет создана папка с названием грамматики (если её не существует),
содержащая наборы негативных и позитивных тестов, разделенных по отдельным файлам.
